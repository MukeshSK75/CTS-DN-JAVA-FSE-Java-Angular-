# Pondicherry Community Event Portal

## Project Overview

The **Pondicherry Community Event Portal** is a responsive web application developed using **HTML5, CSS3, and JavaScript**. It serves as a centralized platform for residents to explore community events, register for activities, provide feedback, view media content, manage preferences, and access location-based event information.

The project demonstrates various HTML5 features, form handling, multimedia integration, client-side storage, event handling, geolocation services, and browser developer tools.

---

## Objectives

* Provide information about upcoming community events.
* Allow residents to register for events online.
* Collect user feedback.
* Demonstrate HTML5 multimedia capabilities.
* Store user preferences using Web Storage APIs.
* Detect user location using Geolocation API.
* Showcase JavaScript event handling and browser DevTools features.

## Features

### 1. Navigation System

* Responsive navigation menu.
* Internal page linking using anchor tags.
* External Help page access.

### 2. Welcome Banner

* Personalized resident welcome section.
* Special offer notifications.
* Styled using CSS classes and inline styling.

### 3. Upcoming Events

* Displays community events with:

  * Event Name
  * Location
  * Date
  * Entry Fee

### 4. Event Photo Gallery

* 2 √ó 3 image gallery layout.
* Image captions.
* Double-click image enlargement functionality.

### 5. Event Registration

* User registration form.
* Required field validation.
* Email validation.
* Date selection.
* Event selection.
* Registration confirmation message.

### 6. Feedback System

* Phone number validation.
* Character counter.
* Dynamic event fee display.
* Feedback submission confirmation.

### 7. Multimedia Support

* HTML5 Video Player.
* Multiple video format support.
* Video status notifications.
* Browser compatibility fallback.

### 8. User Preferences

* Saves preferred event category using Local Storage.
* Session visit tracking using Session Storage.
* Clear preference functionality.

### 9. Geolocation Services

* Detects user location.
* Displays coordinates.
* Handles:

  * Permission Denied
  * Timeout
  * Position Unavailable
  * Unknown Errors

### 10. Chrome DevTools Demonstration

* Console logging examples.
* Network monitoring guide.

* Local Storage inspection guide.
* Responsive testing guide.
