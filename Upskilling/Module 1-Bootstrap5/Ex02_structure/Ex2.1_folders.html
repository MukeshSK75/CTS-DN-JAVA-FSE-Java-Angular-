Bootstrap Downloaded Directory Structure

📁 css/
  • bootstrap.css
  • bootstrap.min.css
  • bootstrap.rtl.css
  • bootstrap-grid.css
  • bootstrap-utilities.css

Contains all compiled CSS. Use .min.css in production for smaller file size. RTL version for right-to-left languages.

📁 js/
  • bootstrap.js
  • bootstrap.min.js
  • bootstrap.bundle.js
  • bootstrap.bundle.min.js
  • bootstrap.esm.js

JS plugins for modals, dropdowns, tooltips. bundle includes Popper.js. Use bundle.min.js for easiest setup.

📁 icons/ (separate package)
  • bootstrap-icons.css
  • bootstrap-icons.woff
  • bootstrap-icons.woff2
  • *.svg files

Over 2000 SVG icons as a separate npm package. Can be used via CSS class or inline SVG.
