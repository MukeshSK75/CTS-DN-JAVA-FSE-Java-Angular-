Bootstrap 5 is a free, open-source CSS framework for building responsive, mobile-first websites. It removed the jQuery dependency from previous versions and ships with a powerful utility class system.

You can install Bootstrap 5 via a CDN link in your HTML, via npm (npm install bootstrap), or by downloading the compiled files from the official website at getbootstrap.com.
