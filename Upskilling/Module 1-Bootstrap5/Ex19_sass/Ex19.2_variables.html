Customizing Bootstrap via _variables.scss

Exercise 19.2 — Override primary color and border radius, then recompile.

Step 1 — Override variables before the Bootstrap import
scss/custom.scss
// 1. Override Bootstrap defaults BEFORE import
$primary:          #7c3aed;   // purple instead of blue
$secondary:        #64748b;
$border-radius:    .75rem;    // larger corners
$border-radius-sm: .25rem;
$border-radius-lg: 1.25rem;
$font-family-base: 'Segoe UI', system-ui, sans-serif;
$font-size-base:   1rem;
$body-bg:          #f8f9fa;

// 2. Import Bootstrap source
@import "../node_modules/bootstrap/scss/bootstrap";
Step 2 — Compile
terminal
# one-time build
npm run build

# watch mode (auto-recompile on save)
npm run watch

# minified output
sass scss/custom.scss dist/css/custom.min.css --style=compressed
Step 3 — Link in HTML
index.html
<!-- Use your compiled file instead of CDN -->
<link rel="stylesheet" href="dist/css/custom.css">
