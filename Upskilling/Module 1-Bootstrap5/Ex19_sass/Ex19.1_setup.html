Bootstrap 5 + Sass Setup

Exercise 19.1 — Setting up a Bootstrap project with Sass using npm

1
Create project folder & init npm
Sets up package.json 
mkdir my-project
cd my-project
npm init -y
2
Install Bootstrap and Sass
Adds both as dependencies 
npm install bootstrap
npm install --save-dev sass
3
Create the project structure
Minimal folder layout 
my-project/
├── scss/
│   └── custom.scss
├── dist/
│   └── (compiled output goes here)
├── index.html
└── package.json
4
Write your custom.scss
Override variables before importing Bootstrap
scss/custom.scss
@import "../node_modules/bootstrap/scss/bootstrap";
5
Add a compile script to package.json
Compiles scss → dist/css
package.json
"scripts": {
  "build": "sass scss/custom.scss dist/css/custom.css",
  "watch": "sass --watch scss/custom.scss dist/css/custom.css"
}
6
Compile
Run once or watch for changes 
npm run build
# or to watch continuously:
npm run watch
7
Link the compiled CSS in index.html
index.html
<link rel="stylesheet" href="dist/css/custom.css">
Tip: Always override variables before the Bootstrap import. Defining them after has no effect since Sass uses first-wins for !default flags.
